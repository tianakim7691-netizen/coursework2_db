1. What the assignment is
    This repository contains focusing on database normalization and advanced SQL querying.

2. How to import the .sql file into MySQL
    - `normalization_process.md`: Detailed steps for 1NF, 2NF, and 3NF.
    - `schema.sql` and `data.sql`: Separate scripts for table creation and data insertion.
    - `queries.sql`: All 12 required SQL statements for Part 4.
    - `exported_database.sql`: A complete backup of the final, populated database (structure and data).

3. Which file contains normalization steps
    The entire final database can be quickly restored using the `exported_database.sql` file.

    The way to handle phpMyAdmin:
        - Create a new, empty database.
        - Select the 'Import' tab.
        - Choose the exported_database.sql file and click 'Go'.

4. Which file contains the queries
    The entire database design and normalization process is documented in `normalization_process.md`

5. Which file contains the Required Queries?
    All required SQL queries (Part 4) are located in the queries.sql file.
    This file contains 12 separate queries covering aggregation, joins, and grouping/filtering
